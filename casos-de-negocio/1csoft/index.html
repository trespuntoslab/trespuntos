Contexto

Una marca con 5 millones de usuarios y sin web propia que lo contara

1CSoft llegó a Tres Puntos sin presencia digital. El producto era robusto —un ERP cloud usado por más de 5 millones de personas en todo el mundo— pero la conversación con el mercado español arrancaba siempre en frío, en llamadas o referencias, sin un canal digital que cualificara antes.

El reto no era estético. Era de arquitectura: cómo ordenar un ERP con 30 funcionalidades de forma que el CEO de una pyme entendiera el valor en segundos, el financiero viera la lógica contable y el técnico pudiera profundizar sin sentirse limitado.

Construimos la web desde la primera línea. Estrategia, contenidos, diseño, desarrollo y conexión con el ERP central. Una landing por cada perfil de buyer, un sistema de bloques editables para que el equipo de 1CSoft pudiera lanzar nuevas piezas sin depender de desarrollo, y formularios conectados directamente al ERP para arrancar pruebas gratuitas en tiempo real.

El reto

Explicar un ERP potente sin abrumar

El software ERP de 1CSoft cubre contabilidad, ventas, compras, almacén, tesorería, fiscalidad y más. Toda esa potencia, sin una arquitectura clara, se convierte en una avalancha de información que el decisor descarta en los primeros segundos.

Y al mismo tiempo: simplificar de más era el otro extremo. El técnico que evalúa el sistema necesita ver que la herramienta es seria, integrable y soporta multiusuario, multipuesto y conexión con la AEAT vía SII.

Las fricciones que detectamos
Sin presencia digital previa La web era el último canal en lanzarse. Todo el inbound dependía de llamadas y referencias.
Producto técnico para audiencia mixta CEO, financiero y técnico tomaban la decisión juntos, pero cada uno mira métricas distintas.
30 funcionalidades en un solo producto Sin una jerarquía clara, el potencial cliente se perdía antes de pedir demo.
Captación dependiente del equipo comercial Cada lead necesitaba intervención manual. La web tenía que cualificar por sí sola.

Preguntas frecuentes

Lo que normalmente nos preguntan

Una web SaaS desde cero para 1CSoft, empresa de software ERP cloud para pymes. El objetivo era transformar visitas en demos y contactos cualificados, con conexión directa al sistema ERP central para activar pruebas en tiempo real.
Arquitectura de información para distintos perfiles (CEO, financiero, técnico), diseño UX/UI orientado a conversión, desarrollo WordPress a medida con bloques flexibles, conexión vía API al ERP central y landings específicas para captación de demos.
El equipo de 1CSoft necesitaba autonomía total para actualizar contenidos, lanzar nuevas funcionalidades y escalar por módulos sin depender de desarrollo. WordPress con bloques personalizados resolvía agilidad editorial; la API conecta cada CTA con el ERP central para arrancar la prueba sin fricción.
Cada formulario de prueba dispara una llamada a la API del ERP central. El sistema provisiona la cuenta automáticamente según el plan elegido (Crecimiento, Avanzado, Corporativo o Autónomo) y devuelve credenciales en segundos. Sin intervención humana hasta que el lead está activo.
Diseño + arquitectura + contenidos: 6 semanas. Desarrollo WordPress + integración con el ERP: 8 semanas. En paralelo, definición de perfiles y plantillas comerciales con el equipo de 1CSoft.
