Contexto del proyecto

Una marca que inspira viajes pero no cierra reservas

Nomade Vans opera en uno de los sectores más emocionales del turismo: el alquiler de furgonetas camper. Su marca visual era potente, sus fotos de carretera generaban deseo. Pero la web no convertía.

El reto era combinar esa emoción con un sistema de reservas claro, accesible y sin fricción. Que la decisión de reservar fuera tan fácil como imaginar el viaje.

El problema

Mucha emoción, poca conversión

Nomade Vans tenía una web visualmente atractiva, pero no estaba pensada para vender.

El usuario se inspiraba, pero no encontraba un camino claro para reservar.

Señales de fricción detectadas
Proceso de reserva difuso El usuario no sabía cómo ni cuándo reservar. Demasiados clics, poca claridad.
Información dispersa Características, precios y disponibilidad repartidos sin jerarquía.
Falta de estructura decisional No había un flujo que guiara de la inspiración a la reserva.
Web sin orientación a conversión Diseño bonito pero sin CTAs claros ni arquitectura de embudo.
La solución

Emoción + estructura = reservas

Replanteamos la web desde cero para que la emoción del viaje conduzca naturalmente a la conversión.
