El proyecto

Una plataforma eSIM global que necesitaba hablar el idioma del negocio

ZIM Connections ofrece planes eSIM de voz y datos en más de 200 destinos. Su producto es sólido: cobertura 5G/4G, activación instantánea, sin roaming, con una app nativa y un Business Suite para empresas.

Pero la web no estaba a la altura del producto. El mensaje se dispersaba entre funcionalidades, el recorrido de usuario no guiaba hacia la acción, y la propuesta de valor quedaba enterrada bajo capas de información.

ZIM necesitaba algo más que un rediseño visual. Necesitaba una arquitectura digital que convirtiera visitas en clientes — tanto viajeros individuales como empresas con flotas de dispositivos.

El reto

Una web que no traducía el valor real del producto

La web existente tenía información, pero no dirección. El usuario llegaba, leía, pero no actuaba. El problema no era solo visual — era estratégico.

Una empresa puede tener un servicio sólido. Pero si la web no lo traduce bien, pierde oportunidades cada día.
